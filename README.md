# NLU-Assignment-2
Problem - 1
learn word embeddings using Word2Vec models trained on textual data collected from IIT Jodhpur sources.

Problem - 2 
Implement and compare different recurrent neural network architectures for character-level name generation

Option 1 — Run in Jupyter Notebook / JupyterLab

Open terminal in the project folder Launch Jupyter: jupyter notebook Open: M25CSA024_Prob1.ipynb/M25CSA024_Prob2.ipynb Click: Kernel → Restart & Run All

Option 2 — Run in Google Colab (Recommended)

Upload the notebook to Colab Enable GPU: Runtime → Change runtime type → GPU Run all cells sequentially

Note: for Problem 1 the data folder with the .txt source file should be present in the same directory to run the code successfully
same is for Problem 2 the directory having the notebook file should have the TrainingNames.txt file to run the code successfully.
